# Sportvereniging H.G.V. Hengelo

H.G.V. Hengelo is een sportvereniging in Hengelo. Deze GitHub-organisatie bevat de broncode van onze websites en interne tools.

---

## Websites

| Repo | Beschrijving |
|------|-------------|
| [hgvhengelo](https://github.com/Sportvereniging-H-G-V/hgvhengelo) | Hoofdwebsite van Sportvereniging H.G.V. Hengelo |
| [apenkooitoernooi](https://github.com/Sportvereniging-H-G-V/apenkooitoernooi) | Website voor apenkooitoernooi.nl |
| [dansstudiohengelo](https://github.com/Sportvereniging-H-G-V/dansstudiohengelo) | Website voor dansstudiohengelo.nl |

| [turnenhengelo](https://github.com/Sportvereniging-H-G-V/turnenhengelo) | Website voor turnenhengelo.nl |

## Tools

| Repo | Beschrijving |
|------|-------------|
| [presentielijst-generator](https://github.com/Sportvereniging-H-G-V/presentielijst-generator) | Privacyvriendelijke webapp om presentielijsten te genereren uit CSV-exports |
| [hgv-signing](https://github.com/Sportvereniging-H-G-V/hgv-signing) | Digitaal ondertekeningsplatform gebaseerd op DocuSeal |

---

## Technologie

De meeste projecten zijn gebouwd met [Astro](https://astro.build), [Tailwind CSS](https://tailwindcss.com) en [TypeScript](https://www.typescriptlang.org).
